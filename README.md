### Планировщик в Банк — Умный задачник в банковском приложении

**Описание:** Интеграция планировщика задач в мобильное приложение банка. Пользователь ставит задачу (купить продукты, забронировать ресторан), получает напоминание и одним кликом переходит в сервис банка/партнера. За выполнение начисляется 5% кэшбэк.

**Роль и вклад:**

- Формализация концепции и бизнес-требований
- Проработка сценариев 4 типов пользователей (родитель, студент, путешественник, предприниматель)
- Спецификация системных функций и интеграций
- Разработка плана релизов с учетом ограничений банковского приложения

**Бизнес-цели:**

- Комиссионный доход 300 руб./мес с активного пользователя
- Снижение оттока пользователей на 15%
- Рост транзакций по картам с 12 до 16 в месяц

**Ключевые функции (Выпуск 1.0):**

- Создание задач с категориями, приоритетами, повторами
- Умные напоминания с переходом к сервисам банка/партнеров
- Автозаполнение заказов и отслеживание статуса
- Автоматическое начисление 5% кэшбэка
- Базовая аналитика использования функций

**Метрики успеха:**

- 30% активных пользователей банка пользуются планировщиком
- 25% задач завершаются переходом в партнерские сервисы
- Создание задачи < 4 кликов
- 98% напоминаний срабатывают корректно
